Django 101
**********

Das *Django 101 Tutorial* basiert auf http://www.django-workshop.de und wurde
für die OpenTechSchool_ PyCoaches_ angepasst. In dem auf vier Stunden angelegten
Tutorial lernst du folgende Dinge über Django:

- Django Setup/Konfiguration
- Models und Object-Relational-Mapping
- Djangos Admin-Interface
- URL Routing
- Views
- Templates
